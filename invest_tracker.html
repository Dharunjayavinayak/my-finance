How this version works

Same backend, improved frontend

Read sync

The app calls your deployed Apps Script with ?action=read and loads three sheet tabs into charts and tables.

Write sync

New rows are sent as JSON using the same structure your old HTML already used: sheet and row.

Autocomplete

Previous names from existing sheet rows are suggested while adding new entries, grouped by the selected category.
